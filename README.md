## 🔥 Universal Prompt

Be concise. Minimize tokens. No repetition. Output only essential information.

## 🧩 High-Performance Prompts

### Debugging
Be concise. No repetition.

Task: Debug this code.

Output:
- Issue
- Fix
- Code snippet only

---

### Code Explanation
Explain this code.

Constraints:
- Max 5 bullet points

---

### Refactoring
Refactor this code.

- Output only improved code

### Architecture

## ⚡ Token Saving Tips

- Do not explain obvious things  
- Avoid repetition  
- Use bullet points  
- Limit output size  
- Always specify scope  

---

## 📉 Before vs After

## 🧠 Golden Rules

- Never load full repo unless needed  
- Always narrow scope  
- Ask step-by-step  
- Iterate instead of dumping everything
