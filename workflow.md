# ⚡ Claude Code Optimization Workflow

## 🧠 Core Principle
Never ask Claude to do everything at once.

---

## 🔄 Step-by-Step Workflow

### 1. Start with Summary
Summarize before deep diving.

Example:
Summarize this codebase in under 100 words.

---

### 2. Narrow Scope
Avoid:
Explain entire repo

Use:
Explain only the authentication module.

---

### 3. Drill Down
Break into steps:
- Overview
- Module
- Function

---

### 4. Iterate
Example flow:
1. Summary
2. Identify issue
3. Fix issue

---

## ⚡ Golden Rules

- Always limit scope
- Avoid full repo context
- Ask step-by-step
- Prefer summaries first

---

## 🚀 Result

- 50–70% fewer tokens
- Faster responses
- Better accuracy
